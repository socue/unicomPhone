### 脚本
- 购买米粉卡时手动选号费时费力，突发奇想弄了个油猴脚本。
- 思路很简单，首先获取待选列表号码，将匹配的号码打印到控制台。

### 申请地址，注意是手机的申请页
##### [米粉卡](https://m.10010.com/scaffold-show/mifanCard?channel=3000)

### 使用方式
1，将该脚本内容(index.js)添加进Tampermonkey中。</br>
2，打开申请页面，找到号码选择的地方如下图。过程需要将浏览器控制台打开，浏览模式为手机模式。</br>
![](img-1.jpg)</br>
3，选择好地址，点击“选择号码”后脚本自动筛选号码并输出到控制台中。</br>
4，无。</br>
